## Описание

Несколько страниц и файлов кода проекта мобильного приложения по оцениванию территорий.

Мобильное приложение делалось как обычный сайт на Nuxt3/Vue3 и TypeScript, после чего оборачивалось в нативное мобильное приложение на Android/iOS с помощью Capacitor. Capacitor также предоставляет различные функции взаимодействия с компонентами телефона (камера, геолокация и пр.).

## Файлы

### index.vue

Главная страница приложения.

### checklists.ts

Сборник всех классов, связанных с работой с чеклистами в мобильном приложении. 

### checklist/[uuid].vue

Страница заполнения чеклиста.
