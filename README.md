# hh-code-examples

Отдельные примеры и вырезки кода из приватных проектов.

## Информационная система

Несколько примеров написанных страниц и компонентов для информационной системы развития территорий АО "ЦЕМРОС".

## Мобильное приложение

Мобильное приложение, оформленное как сайт на Nuxt3/Vue3 и TypeScript, обёрнутое в нативное мобильное приложение на Android/iOS с помощью Capacitor. Также для АО "ЦЕМРОС".

## Личные проекты

У меня также есть несколько открытых репозиториев пет-проектов.

### Сайт настольных игр

Одно из моих хобби - настольные игры. Я написал небольшой статичный сайт со справочником настольных игр для клуба, в котором являюсь администратором.

https://github.com/WhiteRain7/picnic-board-games

Сайт представляет из себя чистый HTML/CSS/JS, выложенный на Github Pages. Данные берутся по токену Google Cloud из открытой Google таблицы. Сайт адаптивен под разные устройства (ПК и телефоны), оптимизирован по объёму (со всеми данными сайт расходует не более 120 Кб трафика), по вычислительным мощностям (достойно работает даже с 20-кратным замедлением процессора). Производительность по метрике Lighthouse - 94.
