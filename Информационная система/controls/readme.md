## Описание

Несколько компонентов из папки "controls", которые позволяют так или иначе вводить данные.

## Файлы

### Calendar.vue

Собственное модальное окно выбора даты (год, месяц и/или день).

### DatePicker.vue

Обёртка для Calendar, поле ввода даты.

```html
<DatePicker />
<DatePicker scope="month" />
```

### Dropdown.vue

Выпадающий список с поддержкой выпадающих подсписков, клавиатурной доступности, а также разделительных черт и пр.

```html
<Dropdown>
    <li>Элемент 1</li>
    <li>Элемент 2</li>
    <li>Элемент 3</li>
</Dropdown>

<Dropdown>
    <li>Элемент 1</li>
    <li>Элемент 2</li>
    <hr/>
    <li>
        <label>Подсписок</label>
        <ul>
            <li>Элемент 3</li>
            <li>Элемент 4</li>
            <li>
                <label>Дополнительный подсписок</label>
                <ul>
                    <li>Элемент 5</li>
                </ul>
            </li>
        </ul>
    </li>
</Dropdown>
```

### TreeElement и TreeSelection

Компонент создания дерева с кликабельными узлами и возможностью сворачивания узлов.
